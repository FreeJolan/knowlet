# Knowlet

> 一个以 Markdown / 纯文本为底座、AI 作为第二大脑的主动式学习与知识沉淀系统。

Knowlet 早期阶段聚焦**学习场景**:OCR 把内容变成可复习的卡片,SRS 驱动节奏,错题反哺笔记。底层是开放的纯文本知识库,AI 是可选增强。

> Knowlet 当前处于设计阶段,代码尚未开始实现。本仓库目前仅维护设计文档与路线图。

## 核心理念

- **AI 是可选增强,不是必需品** —— 无 AI 时仍是可用的笔记库
- **数据主权在用户** —— 本地优先,Markdown + Git
- **能力插件化** —— 各能力模块独立可替换

详见 [ADR-0002 — 三条核心原则](./docs/decisions/0002-core-principles.md)。
