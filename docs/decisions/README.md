# Architecture Decision Records (ADR)

本目录记录 Knowlet 项目重要的架构与产品决策。每条 ADR 一旦合入即视为不可变,后续若被推翻,新增一篇并在新 ADR 中声明 `Supersedes: NNNN`。

## ADR 模板

```markdown
# NNNN — 标题

- Status: Accepted | Proposed | Superseded by NNNN
- Date: YYYY-MM-DD

## Context
做这个决策时面临的问题与背景。

## Decision
最终选择了什么。

## Consequences
带来的好处、代价与后续约束。
```
